# 🗺️ Roadmap — KING Browser v2.0

> Дорожная карта скрытого браузера с обходом блокировок

## 🎯 Концепция KING Browser

**KING** — приватный браузер для России и стран с цензурой. Невидим на демонстрации экрана, обходит блокировки через DoH/прокси, полный контроль через расширенные настройки.

## 🌐 Фаза 3: Обход блокировок

### 3.1 Методы обхода

#### DNS over HTTPS (DoH)
- Cloudflare: `https://cloudflare-dns.com/dns-query`
- Google: `https://dns.google/resolve`
- Quad9: `https://dns.quad9.net/dns-query`
- AdGuard: `https://dns.adguard.com/dns-query`

#### DNS over TLS (DoT)
- Cloudflare: `1.1.1.1:853`
- Google: `8.8.8.8:853`

#### Прокси
- SOCKS5: пользовательский сервер
- HTTP/HTTPS: встроенный список публичных
- Мосты (Bridge): публичные серверы-зеркала

### 3.2 Автоматический режим
```
1. Попытка загрузки страницы → таймаут
2. Если блокировка → попробовать DoH
3. Если DoH не помог → попробовать прокси
4. Если прокси не помог → мост
5. Уведомить пользователя: "Сайт заблокирован, используем обход"
```

### 3.3 Архитектура обхода
```
Services/
  BypassService.cs — НОВЫЙ (оркестрация обхода)
  DohService.cs — НОВЫЙ (DNS over HTTPS)
  ProxyService.cs — НОВЫЙ (управление прокси)
```
