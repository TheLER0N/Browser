# 🐛 Известные проблемы и баги KingBrowser

> Список известных проблем, их влияние и временные решения.  
> Обновляйте этот файл при обнаружении новых проблем.

---

## 🔴 Критические

| ID | Проблема | Влияние | Статус | Workaround |
|----|----------|---------|--------|------------|
| KI-001 | Утечка памяти при закрытии вкладок | Память растёт при большом количестве вкладок | ⚠️ Частично исправлено | Перезапуск браузера при >10 вкладках |
| KI-002 | WebView2 не инициализируется без Runtime | Браузер не запускается | ✅ Исправлено (проверка при запуске) | Установить WebView2 Runtime |
| KI-003 | PrintScreen не блокируется на некоторых клавиатурах | Скриншот возможен через Fn+PrintScreen | 🔄 В работе | Полагаться на WM_PRINTCLIENT блокировку |

## 🟡 Средние

| ID | Проблема | Влияние | Статус | Workaround |
|----|----------|---------|--------|------------|
| KI-004 | Стили в App.xaml (555 строк) — сложно поддерживать | Долгий поиск, конфликты стилей | 📋 Планируется рефакторинг | Искать по ключевым словам |
| KI-005 | DNS тест через UDP может требовать прав администратора | Тест DNS не работает | ✅ Решено | Используйте DoH режим (не требует админ прав) |
| KI-006 | Инкогнито папка не удаляется при краше | Остатки данных на диске | 🔄 Частично исправлено | Ручное удаление `%APPDATA%\GhostBrowser\Incognito\` |
| KI-007 | Нет автотестов | Невозможно проверить регрессию автоматически | 📋 Планируется | Ручное тестирование |
| KI-008 | Нет инсталлятора | Пользователь должен устанавливать .NET и WebView2 вручную | 📋 Планируется | Использовать run.bat |
| KI-009 | Черно-белая тема не переключаемая | Нет выбора темы | 📋 Планируется | Редактировать App.xaml вручную |
