## 8. Правила изменения UI

1. **НЕ менять** StaticResource на HEX внутри ControlTemplate
2. **НЕ удалять** Window.Resources (Storyboard, DataTemplate)
3. **НЕ добавлять** SizeChanged → ApplyClip() без теста
4. **1 файл → сборка → запуск → проверка** перед следующим
5. **Git commit** перед каждым изменением
6. **НЕ использовать** запрещённые цвета (см. начало файла)
7. **SVG Path** вместо Unicode символов для иконок
8. **Плавные анимации** (200-400ms) на все интерактивные элементы

---

## 9. Правила анимаций

| Тип | Длительность | Easing | Описание |
|-----|-------------|--------|----------|
| Hover | 200ms | ease-out | Плавное появление фона |
| Pressed | 100ms | ease-in | Быстрая реакция на клик |
| Fade In | 300-400ms | ease-out | Появление элементов |
| Transition | 250ms | ease-in-out | Переход между состояниями |
| Pulse | 1.5-2s | ease-in-out | Пульсация индикаторов |
| Progress | 500ms | linear | Shimmer эффект |

### Storyboard для hover-эффектов:
```xml
<Trigger Property="IsMouseOver" Value="True">
    <Trigger.EnterActions>
        <BeginStoryboard>
            <Storyboard>
                <DoubleAnimation Storyboard.TargetProperty="Opacity"
                                 To="1" Duration="0:0:0.2">
                    <DoubleAnimation.EasingFunction>
                        <PowerEase EasingMode="EaseOut"/>
                    </DoubleAnimation.EasingFunction>
                </DoubleAnimation>
            </Storyboard>
        </BeginStoryboard>
    </Trigger.EnterActions>
</Trigger>
```
