## Ресурсы
| Ресурс | Назначение | Ответственный |
|--------|------------|---------------|
| .NET 7 SDK | Компиляция | Разработчик |
| WebView2 Runtime 1.0.2739.15 | Рендеринг | Microsoft |
| Windows 10 2004+ | SetWindowDisplayAffinity API | ОС |
| NuGet: Microsoft.Web.WebView2 | Пакет интеграции | Microsoft |

## Риски и митигация
| Риск | Вероятность | Влияние | Меры снижения |
|------|-------------|---------|---------------|
| WebView2 не установлен у пользователя | 🟡 Средняя | 🔴 Высокая | Проверка при запуске, установка рантайма |
| Windows < 10 2004 не поддерживает WDA_EXCLUDEFROMCAPTURE | 🟢 Низкая | 🟡 Средняя | Fallback на WDA_MONITOR |
| Утечки памяти при множестве вкладок | 🟡 Средняя | 🟡 Средняя | Правильный Dispose WebView2 |
| Блокировка антивирусом (перехват клавиш) | 🟡 Средняя | 🔴 Высокая | Подпись кода, white-list |
| Конфликт с другими программами за PrintScreen | 🟢 Низкая | 🟡 Средняя | Graceful degradation |

## Глоссарий
| Термин | Определение |
|--------|-------------|
| Stealth Mode | Режим скрытия окна от программ захвата экрана |
| WDA_EXCLUDEFROMCAPTURE | Флаг Win32 API, скрывающий окно от захвата (Win10 2004+) |
| WebView2 | Chromium-компонент Microsoft для встраивания в приложения |
| MVVM | Model-View-ViewModel паттерн разделения ответственности |
| Panic Button (F12) | Кнопка паники: минимизация + очистка данных |
| UserDataFolder | Изолированное хранилище профиля WebView2 |
| Фингерпринтинг | Техника отслеживания через Canvas/WebGL/Audio API |
