## 🔄 Как идея становится задачей

```
ideas.md (БАНК ИДЕЙ)
    ↓ (пользователь выбирает ID)
tasks.md (TODO — запланировано)
    ↓ (пользователь пишет в task.md "сделай [ID]")
task.md (ТЕКУЩАЯ ЗАДАЧА)
    ↓ (выполнено)
changelog.md (ИСТОРИЯ)
```

**Пользователь может:**
- Написать в `task.md`: "Сделай `STEALTH-001`" → ИИ читает details → делает
- Написать свою задачу → ИИ ищет похожую идею → дополняет план
- Спросить: "Что делать дальше?" → ИИ показывает ТОП идей

## 🔒 Скрытый режим (Stealth)

### 1. 🎭 Призрачный режим (Ghost Mode)
- **ID:** `STEALTH-001`
- **Приоритет:** 🔴 Высокий
- **Описание:** Полная невидимость — окно не видно НИГДЕ: скриншоты, стримы, OBS, Discord, Zoom, Task Manager
- **Технически:** WDA_EXCLUDEFROMCAPTURE + блокировка PrintScreen (GlobalHotkey.cs) + SnippingToolBlocker + ScreenshotBlocker
- **Статус:** ✅ ЗАВЕРШЕНО (Stealth 2.0 — 07.04.2026)
- **Связано с:** `tasks.md` → Stealth 2.0

### 2. 🔐 Паник-кнопка (Panic Button)
- **ID:** `STEALTH-002`
- **Приоритет:** 🔴 Высокий
- **Описание:** F12 → мгновенно сворачивает все вкладки, открывает Google, стирает историю сессии
- **Технически:** Глобальный хук клавиатуры + смена контента + очистка cookies/кэша
- **Статус:** ⏳ Запланировано

### 3. 🕳️ Туннельный режим (Tunnel Mode)
- **ID:** `STEALTH-003`
- **Приоритет:** 🔴 Высокий
- **Описание:** Встроенный VPN/SOCKS5 туннель. Весь трафик через зашифрованный канал
- **Технически:** SOCKS5 клиент + список серверов + авто-подключение при блокировке
- **Статус:** ⏳ Запланировано

### 4. 👤 Фантомные профили (Phantom Profiles)
- **ID:** `STEALTH-004`
- **Приоритет:** 🟡 Средняя
- **Описание:** Временные профили которые полностью стираются при закрытии
- **Технически:** Временный UserDataFolder в RAM-disk + уничтожение папки
- **Статус:** ⏳ Запланировано

### 5. 🎭 Маскировка браузера (Browser Camouflage)
- **ID:** `STEALTH-005`
- **Приоритет:** 🟡 Средняя
- **Описание:** Браузер притворяется другим — Chrome/Safari/Firefox/мобильный
- **Технически:** Кастомный User-Agent + подмена navigator properties
- **Статус:** ⏳ Запланировано

## 🌐 Обход блокировок и свобода

### 6. 🔄 Автоматический обход (Auto-Bypass)
- **ID:** `BYPASS-001`
- **Приоритет:** 🔴 Высокий
- **Описание:** Браузер САМ определяет что сайт заблокирован и пробует: DoH → прокси → мост
- **Технически:** Мониторинг ошибок загрузки + авто-переключение метода обхода
- **Статус:** ⏳ Запланировано

### 7. 🌍 Зеркала сайтов (Mirror Mode)
- **ID:** `BYPASS-002`
- **Приоритет:** 🟡 Средняя
- **Описание:** Автопоиск зеркал заблокированных сайтов + Wayback Machine
- **Технически:** Wayback Machine API + DNS TXT записи + кэширование
- **Статус:** ⏳ Запланировано

### 8. 📡 Децентрализованный веб (Decentralized Web)
- **ID:** `BYPASS-003`
- **Приоритет:** 🟢 Низкая

- **Технически:** IPFS gateway + I2P туннель + Tor мост через SOCKS5
- **Статус:** ⏳ Запланировано

### 9. 🗣️ P2P мессенджер внутри браузера
- **ID:** `BYPASS-004`
- **Приоритет:** 🟢 Низкая
- **Описание:** Встроенный децентрализованный мессенджер (WebRTC + Signal Protocol)
- **Технически:** WebRTC peer-to-peer + шифрование + хранение локально
- **Статус:** ⏳ Запланировано
