# 👻 GhostBrowser — Приватный браузер с режимом невидимости

Современный десктопный браузер на **C# WPF** с функцией скрытия от программ захвата экрана.

## ✨ Возможности

### 🌐 Основное
- **Вкладки** — создавайте, закрывайте и переключайтесь между вкладками
- **Навигация** — назад, вперёд, обновить, домой
- **Умный поиск** — Google, Bing, DuckDuckGo, Яндекс (переключение кнопкой)
- **Закладки** — добавляйте и управляйте закладками
- **История** — автоматическое сохранение посещений
- **Прогресс-бар** — визуальная индикация загрузки

### 👻 Режим невидимости (Stealth Mode)
Скрывает окно от:
- **OBS Studio**
- **Discord (Screen Share)**
- **Zoom**
- **TeamViewer**
- **Программ родительского контроля**
- **Alt+Tab** (частично)

**Горячая клавиша:** `Ctrl+Shift+H`

### 🎨 Дизайн
- Тёмная тема с неоновыми акцентами
- Плавные анимации и свечения
- Современный минималистичный UI
- Красивая страница "Новая вкладка" с быстрыми ссылками

### ⌨️ Горячие клавиши
| Комбинация | Действие |
|------------|----------|
| `Ctrl+T` | Новая вкладка |
| `Ctrl+W` | Закрыть вкладку |
| `Ctrl+L` | Фокус на адресную строку |
| `Ctrl+Shift+H` | Режим невидимости |
| `Alt+←` | Назад |
| `Alt+→` | Вперёд |
| `Ctrl+R` | Обновить |
| `Ctrl+D` | Добавить закладку |
| `Ctrl+1-9` | Переключение вкладок |

## 🚀 Установка и запуск

### Требования
- **Windows 10/11** (версия 2004+)
- **.NET 8 SDK** или **.NET 8 Runtime**
- **WebView2 Runtime** (обычно уже установлен)

### Шаг 1: Установите .NET 8
Скачайте с официального сайта: https://dotnet.microsoft.com/download/dotnet/8.0

Или проверьте установлен ли:
```bash
dotnet --version
```

### Шаг 2: Установите WebView2 Runtime
Скачайте отсюда: https://developer.microsoft.com/ru-ru/microsoft-edge/webview2/

### Шаг 3: Соберите проект
Откройте терминал в папке `GhostBrowser` и выполните:

```bash
dotnet restore
dotnet build
dotnet run
```

### Шаг 4 (опционально): Публикация
Создайте standalone .exe файл:

```bash
dotnet publish -c Release -r win-x64 --self-contained -o ./publish
```

После этого в папке `publish` появится `GhostBrowser.exe`, который можно запускать без .NET SDK.

## 🔧 Как работает режим невидимости

`StealthService` использует Win32 API для модификации стилей окна:

1. **WS_EX_TOOLWINDOW** — окно исключается из Alt+Tab и некоторых скриншотеров
2. **WS_EX_LAYERED + LWA_ALPHA** — установка прозрачности на уровне API
3. **DWMWA_DISALLOW_PEEK** — исключение из предпросмотра Windows
4. **DWMWA_EXCLUDED_FROM_PEEK** — исключение из Aero Peek

> ⚠️ **Важно:** Режим невидимости не гарантирует 100% скрытие от всех программ. Некоторые современные скриншотеры могут всё ещё захватывать окно. Для максимальной приватности используйте в комбинации с VPN.

## 🎯 Как использовать

1. **Запустите браузер**
2. **Введите URL или поисковый запрос** в адресную строку
3. **Переключайте поисковые системы** кнопкой `G` (Google → Bing → DuckDuckGo → Яндекс)
4. **Включите режим невидимости** кнопкой `👻` или `Ctrl+Shift+H`
5. **Добавляйте закладки** кнопкой `☆` или `Ctrl+D`
6. **Смотрите историю** через меню `⋮` → История
